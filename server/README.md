## Quick Start

### 패키지 설치

```bash
$ cd server
$ npm install
```

### 서버 실행

```bash
$ npm run start
```

### 서버 실행 (nodemon; develop 모드)

```bash
$ npm run start:dev
```

### Daemon(트랜잭션 모니터링) 실행

```bash
$ npm run daemon
```
